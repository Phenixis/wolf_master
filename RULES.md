# Rules

## End of the game

The game ends when one of the following conditions is met:
- All werewolves are dead (villagers win)
- The number of alive werewolves is greater than the number of alive villagers (werewolves win)
- The number of alive werewolves is equal to the number of alive villagers, and the mayor is a werewolf (werewolves win)
- The lovers are the last two alive players (lovers win)
- Everyone is dead (no one wins). Example is when the hunter gets killed by the werewolves and kills the last villager in revenge.
